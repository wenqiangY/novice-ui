## 介绍

Novice UI 是一个 基于 Naive UI 二次封装的组件库。支持 naive-ui 的所有功能（Copy ^o^），后续会增加一些增强组件

## 安装

### npm

使用 npm 安装。

```bash
npm i -D Novice-ui
```

### 字体

```bash
npm i -D vfonts
```

### 图标

novice-ui 建议使用 [xicons](https://www.xicons.org) 作为图标库。
